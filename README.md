Modern Legacy Theme for Brackets
===========================
This is a dark theme for Brackets. It is a modern spin on the legacy computer screen.


**Installation**

* In Brackets
* Go to `File > Extension Manager`
* Search for `Modern Legacy`
* Click `Install`
